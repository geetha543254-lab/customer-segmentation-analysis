# Customer Segmentation Analysis

## Project Overview
This project segments customers based on Annual Income and Spending Score using clustering techniques.

## Tools Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Power BI

## Features
- Customer Segmentation
- Spending Pattern Analysis
- Interactive Dashboard
- Data Visualization

## Dashboard Insights
- Total Customers: 200
- Average Income: 60.56k
- Average Spending Score: 50.20
